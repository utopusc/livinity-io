  <action>
**useIsStandalone hook** (use-is-standalone.ts):
Create a hook that returns `true` when the app is running in standalone PWA mode. Check both `navigator.standalone` (iOS) and `window.matchMedia('(display-mode: standalone)').matches` (Android/Chrome). Use `useState` + `useEffect` with a `matchMedia` listener so it reacts if the mode changes. Export as named `useIsStandalone`.

**InstallPromptBanner component** (install-prompt-banner.tsx):
Per user decision: small bottom banner with "Add Livinity to Home Screen" text, app icon, Install button, and Dismiss (X) button. Only shown on mobile when NOT in standalone mode.

Implementation details:
1. **State**: Track `deferredPrompt` (BeforeInstallPromptEvent from Android), `dismissed` (localStorage key `liv:install-prompt-dismissed`), and `isIos` detection (`/iPad|iPhone|iPod/.test(navigator.userAgent) && !('MSStream' in window)`).
2. **beforeinstallprompt listener**: In a `useEffect`, listen for `beforeinstallprompt` event, call `e.preventDefault()`, store the event in state. Clean up on unmount.
3. **Visibility logic**: Show banner when ALL of these are true:
   - `useIsMobile()` returns true
   - `useIsStandalone()` returns false
   - localStorage `liv:install-prompt-dismissed` is NOT set
   - Either `deferredPrompt` exists (Android) OR `isIos` is true
4. **Install button (Android)**: Call `deferredPrompt.prompt()`, await `deferredPrompt.userChoice`, if accepted set localStorage flag and hide.
5. **Install button (iOS)**: Show text "Tap Share then 'Add to Home Screen'" instead of an Install button (iOS has no programmatic install).
6. **Dismiss button**: Set `localStorage.setItem('liv:install-prompt-dismissed', '1')` and hide banner.
7. **Styling**: Fixed bottom banner (above MobileTabBar at z-[70], since tab bar is z-[60]). Use Tailwind: `fixed bottom-[72px] left-4 right-4 z-[70]` with `rounded-xl border border-border-default bg-surface-base shadow-elevation-2 p-3`. Use Framer Motion `AnimatePresence` + `motion.div` for slide-up entrance (initial y=100, animate y=0) and exit.
8. **App icon**: Use the Livinity favicon: `<img src="/favicon/android-chrome-192x192.png" className="h-10 w-10 rounded-xl" />`.
9. Import `useIsMobile` from `@/hooks/use-is-mobile` and `useIsStandalone` from `@/hooks/use-is-standalone`.

Do NOT show on desktop. Do NOT show if already dismissed. Do NOT show in standalone mode.

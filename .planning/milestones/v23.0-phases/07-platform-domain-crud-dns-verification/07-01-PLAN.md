**A. Create DNS verification module at `platform/web/src/lib/dns-verify.ts`:**

This module uses `node:dns/promises` for system resolver checks and `fetch` for Cloudflare DoH cross-check (per CONTEXT.md decision: "DNS resolvers: system resolver + cross-check via Cloudflare 1.1.1.1 DoH").

The relay IP (A record target) is `45.137.194.102` (Server5, per REQUIREMENTS.md).

```typescript
import dns from 'node:dns/promises';

const RELAY_IP = '45.137.194.102';
const CLOUDFLARE_DOH = 'https://1.1.1.1/dns-query';

export interface DnsCheckResult {
  aRecordVerified: boolean;
  txtRecordVerified: boolean;
  aRecordValues: string[];
  txtRecordValues: string[];
  error?: string;
}

export async function checkARecord(domain: string): Promise<{ verified: boolean; values: string[] }> {
  // System resolver check
  try {
    const addresses = await dns.resolve4(domain);
    const verified = addresses.includes(RELAY_IP);
    return { verified, values: addresses };
  } catch (err: any) {
    return { verified: false, values: [] };
  }
}

export async function checkARecordDoH(domain: string): Promise<{ verified: boolean; values: string[] }> {
  // Cloudflare DoH cross-check
  try {
    const url = `${CLOUDFLARE_DOH}?name=${encodeURIComponent(domain)}&type=A`;
    const res = await fetch(url, {
      headers: { 'Accept': 'application/dns-json' },
    });
    if (!res.ok) return { verified: false, values: [] };
    const data = await res.json() as { Answer?: Array<{ type: number; data: string }> };
    const addresses = (data.Answer || []).filter(a => a.type === 1).map(a => a.data);
    const verified = addresses.includes(RELAY_IP);
    return { verified, values: addresses };
  } catch {
    return { verified: false, values: [] };
  }
}

export async function checkTxtRecord(domain: string, expectedToken: string): Promise<{ verified: boolean; values: string[] }> {
  const txtHost = `_livinity-verification.${domain}`;
  const expectedValue = `liv_verify=${expectedToken}`;
  try {
    const records = await dns.resolveTxt(txtHost);
    // TXT records return 2D array -- join chunks per record
    const flatRecords = records.map(chunks => chunks.join(''));
    const verified = flatRecords.some(r => r === expectedValue);
    return { verified, values: flatRecords };
  } catch {
    return { verified: false, values: [] };
  }
}

export async function verifyDomainDns(domain: string, verificationToken: string): Promise<DnsCheckResult> {
  // Run system resolver A check + Cloudflare DoH A check + TXT check in parallel
  const [aSystem, aDoH, txt] = await Promise.all([
    checkARecord(domain),
    checkARecordDoH(domain),
    checkTxtRecord(domain, verificationToken),
  ]);

  // A record passes if EITHER system resolver OR Cloudflare DoH confirms it
  // (handles DNS propagation where one resolver has it and other doesn't yet)
  const aRecordVerified = aSystem.verified || aDoH.verified;
  const allAValues = [...new Set([...aSystem.values, ...aDoH.values])];

  return {
    aRecordVerified,
    txtRecordVerified: txt.verified,
    aRecordValues: allAValues,
    txtRecordValues: txt.values,
  };
}

export function generateVerificationToken(): string {
  // crypto.randomBytes(32).toString('hex') per CONTEXT.md decision
  const crypto = require('node:crypto');
  return crypto.randomBytes(32).toString('hex');
}

export const RELAY_SERVER_IP = RELAY_IP;
```

**B. Create `platform/web/src/app/api/domains/route.ts` (GET list + POST create):**

Auth: Use the same `getUser` pattern from dashboard route (read `liv_session` cookie, call `getSession`).

GET /api/domains:
- Auth required
- Query `custom_domains` WHERE `user_id = userId` ORDER BY `created_at DESC`
- Return JSON array of domain objects

POST /api/domains:
- Auth required
- Body: `{ domain: string }`
- Validate domain format (basic regex: must have at least one dot, no protocol prefix, alphanumeric + hyphens + dots)
- Check domain uniqueness across ALL users (SELECT WHERE domain = $1)
- Check user's domain count < 3 (free tier limit per CONTEXT.md)
- Generate verification token via `crypto.randomBytes(32).toString('hex')`
- Insert into `custom_domains` with status `pending_dns`
- Return the created domain record with DNS instructions (relay IP + TXT record instructions)

**C. Create `platform/web/src/app/api/domains/[id]/route.ts` (GET detail + DELETE):**

GET /api/domains/[id]:
- Auth required
- Query single domain by id AND user_id
- Return 404 if not found
- Return domain object with full details

DELETE /api/domains/[id]:
- Auth required
- Delete domain by id AND user_id (ensures ownership)
- Return 200 on success, 404 if not found

**D. Create `platform/web/src/app/api/domains/[id]/verify/route.ts` (POST trigger verification):**

POST /api/domains/[id]/verify:
- Auth required
- Fetch domain by id AND user_id
- Call `verifyDomainDns(domain.domain, domain.verification_token)`
- Update domain record: set `dns_a_verified`, `dns_txt_verified`, `last_dns_check = NOW()`
- If both A + TXT verified: set `status = 'dns_verified'`, `verified_at = NOW()`
- If previously verified but now failed: set `status = 'dns_changed'`
- If not verified: keep `status = 'pending_dns'` (or set to `dns_failed` if past 48h since created_at)
- Return updated domain object with verification result details

All API routes use Drizzle ORM (`import { db } from '@/lib/drizzle'` and `import { customDomains } from '@/db/schema'`), and session auth via the `getUser` helper pattern (cookie -> getSession).

| T-63-05-02 | Spoofing | OWUI's user gets Bolt.diy's broker_usage rows due to wrong api_key_id | mitigate | Task 4 cross-references broker_usage.api_key_id against api_keys row for `phase-63-open-webui`; mismatch → Phase 62 hot-patch |

| T-63-05-04 | Denial-of-Service | OWUI's default polling hits /v1/models too aggressively (rate limit) | accept | Default OWUI polling is on-demand on user action; well below Phase 60 baseline |

    Implementation details (follow exactly):

    1. **listDir(containerName, dirPath)** ŌĆö uses `docker exec`:
       - Validate `dirPath` starts with `/` (reject with `[bad-path]` error if not absolute) ŌĆö paths are interpreted INSIDE the container, not host.
       - Run `exec` with `Cmd: ['ls', '-la', '--time-style=+%s', '--', dirPath]`, `Tty: false`, `AttachStdout: true`, `AttachStderr: true`, `AttachStdin: false`.
       - `stream = await exec.start({hijack: true, stdin: false})`. Collect all chunks, concat to Buffer, decode with `stripDockerStreamHeaders(buffer)`.
       - After stream ends: `const info = await exec.inspect()`. If `info.ExitCode !== 0`, throw `Error('[ls-failed] ' + decoded.trim())` ŌĆö surfaces "no such file" / "permission denied" clearly.
       - Parse the ls output. Lines look like: `drwxr-xr-x 2 root root 4096 1713024000 foo`. First line is the `total N` summary ŌĆö skip it. Split each line on whitespace with `line.trim().split(/\s+/, 9)` so trailing filename with spaces survives ŌĆö take the filename as `line.slice(indexOfEighthFieldEnd).trim()`, or simpler: `parts.slice(8).join(' ')`.
         - mode = parts[0]
         - size = parseInt(parts[4], 10)
         - mtime = parseInt(parts[5], 10)
         - rawName = parts.slice(6).join(' ').replace(/^[0-9]+\s+/, '') ŌĆö actually parts index depends on ls output. USE THIS FORMULA INSTEAD, which is robust against name-with-spaces:
           ```
           const match = line.match(/^(\S+)\s+\d+\s+\S+\s+\S+\s+(\d+)\s+(\d+)\s+(.+)$/)
           if (!match) continue
           const [, mode, sizeStr, mtimeStr, rest] = match
           ```
         - For symlinks, `rest` is `link -> target`. Split on ` -> ` to separate name and target.
         - Skip `.` and `..` entries.
         - type: `mode[0]==='d'`ŌåÆ'dir', `mode[0]==='-'`ŌåÆ'file', `mode[0]==='l'`ŌåÆ'symlink', else 'other'.
       - Return sorted: directories first, then files, each alpha-ascending.

    2. **readFile(containerName, filePath, maxBytes=1_000_000)** ŌĆö size check then cat:
       - Validate absolute path (`startsWith('/')`).
       - First exec `Cmd: ['stat', '-c', '%s', '--', filePath]`, `Tty: false`. Parse integer. If `size > maxBytes`, throw `Error('[file-too-large] file is ' + size + ' bytes, max ' + maxBytes)`. This surfaces as BAD_REQUEST in the tRPC route.
       - Second exec `Cmd: ['cat', '--', filePath]`, `Tty: false`. Concat chunks, decode with `stripDockerStreamHeaders`. Return `{content: decoded, size}`.
       - If cat exits non-zero, throw `Error('[read-failed] ' + stderr)`.

    3. **writeFile(containerName, filePath, content)** ŌĆö tar + putArchive:
       - Validate absolute path.
       - Build a single-file tar in memory using `archiver` (already installed): `const archive = archiver('tar'); const chunks: Buffer[] = []; archive.on('data', c => chunks.push(c));`. Append: `archive.append(Buffer.isBuffer(content) ? content : Buffer.from(content, 'utf-8'), {name: basename(filePath), mode: 0o644})`. `await archive.finalize()`. Concat buffers.
       - `await container.putArchive(tarBuffer, {path: dirname(filePath)})`. Dockerode accepts `Buffer` directly (it's treated as a raw tar payload).
       - `dirname(filePath)` is `path.posix.dirname(filePath)` ŌĆö import `posix` from `node:path` because container paths are POSIX even on Windows hosts.
       - If putArchive rejects with statusCode 404, throw `Error('[dir-not-found] ' + dirname(filePath))`.

    4. **downloadArchive(containerName, path)** ŌĆö streaming tar:
       - Validate absolute path.
       - `const stream = await container.getArchive({path})` ŌĆö returns `NodeJS.ReadableStream` (dockerode v4 types are loose; cast via `as unknown as NodeJS.ReadableStream`).
       - Return the stream directly. Caller is the REST endpoint which pipes to the HTTP response with `Content-Type: application/x-tar`.
       - If getArchive rejects with statusCode 404, throw `Error('[not-found] ' + path)`.

    5. **deleteFile(containerName, path, recursive)** ŌĆö docker exec rm:
       - Validate absolute path.
       - Reject `path === '/'` and `path === ''` (belt-and-braces; exec rm -rf / on a container is still bad UX). Throw `[bad-path] refuses to delete root`.
       - Exec: `Cmd: recursive ? ['rm', '-rf', '--', path] : ['rm', '--', path]`, `Tty: false`.
       - If exec exit code !== 0, throw `Error('[delete-failed] ' + stderr)`. This surfaces "directory not empty" when caller forgot `recursive: true`.

    Shared helper for non-TTY exec (to avoid duplicating the collect-stream-and-inspect loop): write a local `async function execCapture(container, cmd): Promise<{stdout: string; stderr: string; exitCode: number}>`. Exec with `AttachStderr: true`, collect output buffer, but be aware: when `Tty: false` AND `AttachStderr: true`, Docker uses multiplexed stream ŌĆö stdout and stderr are interleaved with the 8-byte frame header. Each frame's byte 0 tells you the stream (1=stdout, 2=stderr). You already have `stripDockerStreamHeaders` for stdout-only decoding, but here we need to demultiplex. Copy the demux loop from `stripDockerStreamHeaders` but branch on `buffer[offset]` to route bytes to the right string.

    ```typescript
    // In container-files.ts ŌĆö DO NOT modify stripDockerStreamHeaders, this is a specialized variant.
    function demuxDockerStream(buf: Buffer): {stdout: string; stderr: string} {
      let stdout = ''
      let stderr = ''
      let offset = 0
      while (offset + 8 <= buf.length) {
        const streamType = buf[offset] // 1=stdout, 2=stderr
        const frameSize = buf.readUInt32BE(offset + 4)
        offset += 8
        if (offset + frameSize > buf.length) break
        const frame = buf.subarray(offset, offset + frameSize).toString('utf-8')
        if (streamType === 2) stderr += frame; else stdout += frame
        offset += frameSize
      }
      return {stdout, stderr}
    }
    ```

    // Container file download ŌĆö CFB-02
    // Returns raw tar stream from docker getArchive. UI unpacks client-side with js-untar / unpack OR
    // server helper: see Plan 18-02 notes. Tar is always a single-entry tar for a file download;
    // for a directory path, it's a multi-entry tar of the whole tree.

        // Auth: session token cookie (same as /api/desktop/resize)

        if (!isValid) return response.status(401).json({error: 'unauthorized'})

        const name = request.params.name
        const path = typeof request.query.path === 'string' ? request.query.path : ''
        if (!name || !path.startsWith('/')) {
          return response.status(400).json({error: 'name required and path must be absolute'})
        }

          stream = await downloadContainerArchive(name, path)
        } catch (err: any) {

          if (err.message?.includes('[bad-path]'))  return response.status(400).json({error: err.message.replace('[bad-path] ', '')})
          throw err

        // Content type is tar; UI treats it as a binary blob and unpacks on the client.

        this.logger.error(`Container download error`, err)
        if (!response.headersSent) response.status(500).json({error: err.message})

    Implementation:

    ```typescript
    import Busboy from 'busboy' // top of file, alphabetical with other imports

    // Container file upload ŌĆö CFB-03
    // multipart/form-data; single "file" field + optional filename override in query.
    this.app.post('/api/docker/container/:name/file', async (request, response) => {
      try {
        const sessionToken = request?.cookies?.LIVINITY_SESSION
        if (!sessionToken) return response.status(401).json({error: 'unauthorized'})
        const isValid = await this.verifyToken(sessionToken).catch(() => false)
        if (!isValid) return response.status(401).json({error: 'unauthorized'})

        const name = request.params.name
        const dirPath = typeof request.query.path === 'string' ? request.query.path : ''
        if (!name || !dirPath.startsWith('/')) {
          return response.status(400).json({error: 'name required and path must be absolute directory'})
        }

        const contentType = request.headers['content-type'] || ''
        if (!contentType.startsWith('multipart/form-data')) {
          return response.status(400).json({error: 'Content-Type must be multipart/form-data'})
        }

        // Stream the single file into a Buffer (capped at 110MB to match CFB acceptance of 100MB)
        const MAX_UPLOAD_BYTES = 110 * 1024 * 1024
        const bb = Busboy({headers: request.headers, limits: {files: 1, fileSize: MAX_UPLOAD_BYTES}})
        let fileBuffer: Buffer | null = null
        let fileName: string | null = null
        let truncated = false

        const finished = new Promise&lt;void&gt;((resolve, reject) => {
          bb.on('file', (_fieldName, stream, info) => {
            fileName = info.filename
            const chunks: Buffer[] = []
            stream.on('data', (c: Buffer) => chunks.push(c))
            stream.on('limit', () => { truncated = true })
            stream.on('end', () => { fileBuffer = Buffer.concat(chunks) })
            stream.on('error', reject)
          })
          bb.on('finish', () => resolve())
          bb.on('error', reject)
        })

        request.pipe(bb)
        await finished

        if (truncated) return response.status(413).json({error: `file exceeds ${MAX_UPLOAD_BYTES} bytes`})
        if (!fileBuffer || !fileName) return response.status(400).json({error: 'no file in upload'})

        // Construct full container path = dirPath + '/' + fileName (sanitize ŌĆö strip any '/' in fileName)
        const safeName = fileName.replace(/[\\/]/g, '_')
        const targetPath = dirPath.endsWith('/') ? `${dirPath}${safeName}` : `${dirPath}/${safeName}`

        try {
          await writeContainerFile(name, targetPath, fileBuffer)
        } catch (err: any) {
          if (err.message?.includes('[not-found]'))     return response.status(404).json({error: err.message.replace('[not-found] ', '')})
          if (err.message?.includes('[dir-not-found]')) return response.status(404).json({error: err.message.replace('[dir-not-found] ', '')})
          if (err.message?.includes('[bad-path]'))      return response.status(400).json({error: err.message.replace('[bad-path] ', '')})
          throw err
        }

        return response.json({success: true, path: targetPath, bytes: fileBuffer.length})
      } catch (err: any) {
        this.logger.error(`Container upload error`, err)
        if (!response.headersSent) response.status(500).json({error: err.message})
      }
    })
    ```

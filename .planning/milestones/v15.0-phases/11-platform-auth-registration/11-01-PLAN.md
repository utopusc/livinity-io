**2. nexus/packages/core/src/sdk-agent-runner.ts** (add robustness):

Add three safety mechanisms inside the `run()` method:

a) **Stream read watchdog** (Pitfall 2 from research): Add a watchdog that aborts if no SDK message arrives within 60 seconds. Place this around the `for await (const message of messages)` loop:

```typescript
// Inside run(), before the for-await loop:
const abortController = new AbortController();
let lastMessageTime = Date.now();

// Watchdog: abort if no message received within 60s (handles silent connection death)
const watchdog = setInterval(() => {
  if (Date.now() - lastMessageTime > 60_000) {
    logger.warn('SdkAgentRunner: watchdog triggered — no message in 60s, aborting');
    abortController.abort();
  }
}, 10_000);
```

Update the `query()` call to pass `abortController.signal`:
```typescript
const messages = query({
  prompt: task,
  options: {
    // ... existing options ...
    abortController,  // Pass abort controller for watchdog + external cancellation
  },
});
```

Inside the for-await loop, update `lastMessageTime` on every received message:
```typescript
for await (const message of messages) {
  lastMessageTime = Date.now(); // Reset watchdog on each message
  // ... rest of existing message handling ...
}
```

In the finally block (add one wrapping the try/catch), clear the watchdog:
```typescript
} finally {
  clearInterval(watchdog);
}
```

b) **Max budget enforcement** (Pitfall 9): Add `maxBudgetUsd` to the query options. Set it based on tier:
```typescript
const budgetByTier: Record<string, number> = {
  opus: 10.0,
  sonnet: 5.0,
  haiku: 2.0,
  flash: 2.0,
};
const maxBudgetUsd = budgetByTier[tier] ?? 5.0;
```
Add `maxBudgetUsd` to the query options object.

c) **Minimal subprocess environment** (Pitfall 12): Add an `env` option to query to avoid leaking secrets:
```typescript
const safeEnv: Record<string, string> = {
  HOME: process.env.HOME || '/root',
  PATH: process.env.PATH || '/usr/local/bin:/usr/bin:/bin',
  NODE_ENV: process.env.NODE_ENV || 'production',
  LANG: process.env.LANG || 'en_US.UTF-8',
  // ANTHROPIC_API_KEY is handled by the SDK internally
};
if (process.env.ANTHROPIC_API_KEY) {
  safeEnv.ANTHROPIC_API_KEY = process.env.ANTHROPIC_API_KEY;
}
```
Add `env: safeEnv` to the query options.

IMPORTANT: Do NOT change the existing event emission logic, the message type handling (assistant/result), the tool call tracking, the TTFB measurement, or the AgentResult return shape. These are all correct and consumed by the SSE endpoint in api.ts and the livinityd AI module.

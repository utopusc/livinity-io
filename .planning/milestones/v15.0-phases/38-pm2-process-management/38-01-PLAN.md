
2. `PM2ProcessInfo` interface:
   ```typescript
   export interface PM2ProcessInfo {
     name: string
     pm_id: number
     status: string        // 'online' | 'stopped' | 'errored' | 'launching'
     cpu: number            // percentage (0-100)
     memory: number         // bytes
     uptime: number         // milliseconds since process started (0 if stopped)
     restarts: number       // restart count
     isProtected: boolean   // true if name matches PROTECTED_PM2_PROCESSES
   }
   ```

3. `PM2ProcessDetail` interface:
   ```typescript
   export interface PM2ProcessDetail {
     name: string
     pm_id: number
     pid: number            // OS process ID (0 if stopped)
     script: string         // script path
     cwd: string            // working directory
     nodeVersion: string    // e.g. "v20.11.0" or "N/A"
     execMode: string       // "fork" or "cluster"
     status: string
     restarts: number
     uptime: number         // milliseconds
     createdAt: number      // unix timestamp ms
   }
   ```

4. `PM2Operation` type:
   ```typescript
   export type PM2Operation = 'start' | 'stop' | 'restart'
   ```

Create `modules/pm2/pm2.ts` with:

1. Import `$` from `execa` (codebase convention -- NOT `execa` named export; use `$` tagged template).

2. `isProtectedProcess(name: string): boolean` -- checks if name is in PROTECTED_PM2_PROCESSES (exact match, case-insensitive).

3. `listProcesses(): Promise<PM2ProcessInfo[]>`:
   - Execute `$\`pm2 jlist\`` with `{ reject: false }` option
   - Parse stdout as JSON array
   - Map each entry to PM2ProcessInfo:
     - `name`: entry.name
     - `pm_id`: entry.pm_id
     - `status`: entry.pm2_env?.status ?? 'stopped'
     - `cpu`: entry.monit?.cpu ?? 0
     - `memory`: entry.monit?.memory ?? 0
     - `uptime`: entry.pm2_env?.status === 'online' ? (Date.now() - (entry.pm2_env?.pm_uptime ?? Date.now())) : 0
     - `restarts`: entry.pm2_env?.restart_time ?? 0
     - `isProtected`: isProtectedProcess(entry.name)
   - If pm2 jlist fails or returns empty, return empty array (do NOT throw -- PM2 might not have processes)
   - Sort by pm_id ascending

4. `manageProcess(name: string, operation: PM2Operation): Promise<{success: boolean; message: string}>`:
   - Check `isProtectedProcess(name)` for 'stop' operation -- throw `[protected-process] Cannot stop protected process: ${name}`
   - Execute `$\`pm2 ${operation} ${name}\``
   - Return `{ success: true, message: \`Process ${name} ${operation}ed successfully\` }`
   - Fix grammar: 'stop' -> 'stopped' (not 'stoped')

5. `getProcessLogs(name: string, lines: number = 200): Promise<string>`:
   - Execute `$\`pm2 logs ${name} --lines ${String(lines)} --nostream --raw\`` with `{ reject: false }`
   - Return stdout (or empty string if command fails)
   - Timeout: 10 seconds (pass `{ timeout: 10000, reject: false }`)

6. `describeProcess(name: string): Promise<PM2ProcessDetail>`:
   - Execute `$\`pm2 describe ${name} --json\`` (note: outputs a JSON array, take first element)
   - Parse first element from JSON array
   - Map to PM2ProcessDetail:
     - `name`: entry.name
     - `pm_id`: entry.pm_id
     - `pid`: entry.pid ?? 0
     - `script`: entry.pm2_env?.pm_exec_path ?? 'N/A'
     - `cwd`: entry.pm2_env?.pm_cwd ?? 'N/A'
     - `nodeVersion`: entry.pm2_env?.node_version ?? 'N/A'
     - `execMode`: entry.pm2_env?.exec_mode ?? 'fork'
     - `status`: entry.pm2_env?.status ?? 'stopped'
     - `restarts`: entry.pm2_env?.restart_time ?? 0
     - `uptime`: entry.pm2_env?.status === 'online' ? (Date.now() - (entry.pm2_env?.pm_uptime ?? Date.now())) : 0
     - `createdAt`: entry.pm2_env?.created_at ?? 0
   - If JSON parse fails or no entries, throw `[not-found] Process not found: ${name}`

3. Export default router with four routes:

   **pm2.list** (query):
   - No input required
   - Call `listProcesses()`
   - Return array of PM2ProcessInfo

   **pm2.manage** (mutation):
   - Input: `z.object({ name: z.string().min(1).max(255), operation: z.enum(['start', 'stop', 'restart']) })`
   - Call `manageProcess(input.name, input.operation)`
   - Catch errors: if message includes `[protected-process]`, throw TRPCError FORBIDDEN (same pattern as Docker protected containers)
   - Otherwise throw INTERNAL_SERVER_ERROR

   **pm2.logs** (query):
   - Input: `z.object({ name: z.string().min(1).max(255), lines: z.number().min(10).max(2000).optional().default(200) })`
   - Call `getProcessLogs(input.name, input.lines)`

   **pm2.describe** (query):
   - Input: `z.object({ name: z.string().min(1).max(255) })`
   - Call `describeProcess(input.name)`
   - Catch errors: if message includes `[not-found]`, throw TRPCError NOT_FOUND (same pattern as Docker)
   - Otherwise throw INTERNAL_SERVER_ERROR

Output: Updated `use-app-store-bridge.ts` with progress polling, credentials forwarding, installing status, and corrected reportEvent URL.

In `livos/packages/ui/src/hooks/use-app-store-bridge.ts`, rewrite the `handleInstall` callback:

1. **Send installing status immediately** before calling install mutate:
   ```typescript
   sendToIframe({type: 'progress', appId, progress: 0})
   ```
   And send a status update with `installing`:
   ```typescript
   sendToIframe({type: 'status', apps: [{id: appId, status: 'installing', progress: 0}]})
   ```
   Note: this is a partial status update just for the installing app. The full status list is sent after install completes.

2. **Start a polling interval** that queries `apps.state` every 2 seconds and forwards progress:
   ```typescript
   const pollInterval = setInterval(async () => {
     try {
       const stateResult = await trpcClient.apps.state.query({appId})
       if (stateResult.progress > 0) {
         sendToIframe({type: 'progress', appId, progress: stateResult.progress})
       }
       // Stop polling if no longer installing
       if (stateResult.state !== 'installing') {
         clearInterval(pollInterval)
       }
     } catch {
       // Ignore polling errors
     }
   }, 2000)
   ```

3. **Call the install mutation** (existing `trpcClient.apps.install.mutate({appId})`). This blocks until install completes.

4. **Clear the polling interval** after install returns (in case it hasn't stopped):
   ```typescript
   clearInterval(pollInterval)
   ```

5. **Send 100% progress**:
   ```typescript
   sendToIframe({type: 'progress', appId, progress: 100})
   ```

6. **Fetch credentials after successful install**. After `reportEvent` and before sending `installed`, query the apps list and extract credentials for the installed app:
   ```typescript
   // Fetch credentials for the newly installed app
   try {
     const appsList = await trpcClient.apps.list.query()
     const installedApp = appsList.find((a) => a.id === appId && !('error' in a))
     if (installedApp && 'credentials' in installedApp && installedApp.credentials) {
       const {defaultUsername, defaultPassword} = installedApp.credentials
       if (defaultUsername || defaultPassword) {
         sendToIframe({
           type: 'credentials',
           appId,
           username: defaultUsername || '',
           password: defaultPassword || '',
         })
       }
     }
   } catch {
     // Ignore credentials fetch errors
   }
   ```

7. **Full handleInstall** should follow this order:
   a. Send progress 0 + installing status to iframe
   b. Start polling interval
   c. `await trpcClient.apps.install.mutate({appId})`
   d. Clear polling interval
   e. Send progress 100
   f. `reportEvent(appId, 'install')`
   g. Fetch and send credentials
   h. `sendToIframe({type: 'installed', appId, success: true})`
   i. (catch block sends `{type: 'installed', appId, success: false, error: message}` and clears interval)
   j. (finally block) `await sendStatusToIframe()` + invalidate queries

8. **Desktop refresh** (R-STORE-REFRESH): The existing `utilsRef.current.apps.list.invalidate()` and `utilsRef.current.apps.state.invalidate()` calls are already present. Ensure they remain in the finally/after block. This triggers React Query refetch which updates the desktop grid.

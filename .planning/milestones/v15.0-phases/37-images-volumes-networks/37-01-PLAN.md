**types.ts** -- Append these new interfaces after the existing ContainerStats interface:

```typescript
export interface ImageInfo {
  id: string           // first 12 chars of image ID (strip sha256: prefix)
  repoTags: string[]   // e.g. ["nginx:latest", "nginx:1.25"]
  size: number          // bytes
  created: number       // unix timestamp
}

export interface VolumeInfo {
  name: string
  driver: string        // e.g. "local"
  mountpoint: string    // host path
  createdAt: string     // ISO date string
}

export interface NetworkInfo {
  id: string            // first 12 chars
  name: string
  driver: string        // e.g. "bridge", "host", "overlay"
  scope: string         // "local" | "swarm" | "global"
  containerCount: number
}

export interface NetworkContainer {
  name: string
  ipv4: string
  macAddress: string
}

export interface NetworkDetail {
  id: string
  name: string
  driver: string
  scope: string
  containers: NetworkContainer[]
}
```

**docker.ts** -- Add import for the new types and add these 7 domain functions after the existing getContainerStats function. All use the existing `docker` singleton:

1. `listImages()`: Call `docker.listImages()`. Map each image to ImageInfo. Strip `sha256:` prefix from Id, take first 12 chars. Handle images with no RepoTags by defaulting to `["<none>:<none>"]`. Sort by created descending (newest first).

2. `removeImage(id: string, force: boolean)`: Call `docker.getImage(id).remove({force})`. Return `{success: true, message: "Image removed successfully"}`. Wrap in try/catch -- if statusCode 404 throw `[not-found] Image not found: ${id}`, if statusCode 409 throw `[in-use] Image is in use by a container` (new error tag pattern).

3. `pruneImages()`: Call `docker.pruneImages()`. The Docker API returns `{ImagesDeleted: [...], SpaceReclaimed: number}`. Return `{spaceReclaimed: result.SpaceReclaimed || 0, deletedCount: (result.ImagesDeleted || []).length}`.

4. `listVolumes()`: Call `docker.listVolumes()`. Docker returns `{Volumes: [...], Warnings: [...]}`. Map each volume to VolumeInfo. CreatedAt comes from `volume.CreatedAt` (ISO string) or empty string. Sort alphabetically by name.

5. `removeVolume(name: string)`: Call `docker.getVolume(name).remove()`. Return `{success: true, message: "Volume '${name}' removed successfully"}`. If statusCode 404 throw `[not-found] Volume not found: ${name}`, if statusCode 409 throw `[in-use] Volume is in use`.

6. `listNetworks()`: Call `docker.listNetworks()`. Map each to NetworkInfo. containerCount = `Object.keys(network.Containers || {}).length`. Sort by name alphabetically. Id is first 12 chars.

7. `inspectNetwork(id: string)`: Call `docker.getNetwork(id).inspect()`. Map to NetworkDetail. For each container in Containers object, extract name (strip leading `/`), IPv4Address, MacAddress. If statusCode 404 throw `[not-found] Network not found: ${id}`.

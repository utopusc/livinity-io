**A. Modify `shared/dialog.ts` -- mobile-optimized dialog padding and sizing:**

The current `dialogContentClass` has `p-8` (32px padding) which wastes 64px of horizontal space on mobile. Change the padding to be responsive:

Replace `p-8` with `p-5 sm:p-8` in `dialogContentClass`. This gives 20px padding on mobile (< 640px) and 32px on larger screens.

Also change `rounded-24` to `rounded-20 sm:rounded-24` for tighter border radius on mobile.

Also change `max-h-[calc(100%-16px)]` to `max-h-[calc(100%-8px)] sm:max-h-[calc(100%-16px)]` for slightly more vertical space on mobile.

The updated `dialogContentClass` should be:
```typescript
export const dialogContentClass = tw`fixed left-[50%] top-[50%] z-[500] flex flex-col translate-x-[-50%] translate-y-[-50%] gap-5 sm:gap-6 rounded-20 sm:rounded-24 bg-dialog-content/75 contrast-more:bg-dialog-content p-5 sm:p-8 shadow-dialog transform-gpu backdrop-blur-lg md:backdrop-blur-2xl contrast-more:backdrop-blur-none duration-200 outline-none max-h-[calc(100%-8px)] sm:max-h-[calc(100%-16px)] border border-border-default`
```

Then in `dialog.tsx`, the `DialogContent` component applies `'w-full max-w-[calc(100%-40px)] sm:max-w-[480px]'`. Change this to:
`'w-full max-w-[calc(100%-24px)] sm:max-w-[480px]'`

This reduces the horizontal margin from 20px per side to 12px per side on mobile, giving substantially more content width. On a 375px screen: 375 - 24 - 40 = 311px content (was 271px).

**B. Modify `settings-toggle-row.tsx` -- 44px minimum touch target:**

The toggle row needs a minimum height for reliable touch interaction. Change the outer div class:

From: `'flex items-center justify-between rounded-radius-md border border-border-default bg-surface-base p-4'`
To: `'flex items-center justify-between rounded-radius-md border border-border-default bg-surface-base p-4 min-h-[52px]'`

The 52px accounts for p-4 (16px top + bottom) plus content, ensuring the total touch area exceeds 44px.

Also ensure the Switch component has an adequate touch target by wrapping it in a padding area. Change:
```tsx
<Switch checked={checked} onCheckedChange={onCheckedChange} disabled={disabled} />
```
To:
```tsx
<div className='flex items-center justify-center min-h-[44px] min-w-[44px]'>
  <Switch checked={checked} onCheckedChange={onCheckedChange} disabled={disabled} />
</div>
```

This gives the Switch a 44x44px touch zone even though the visual Switch is only 20x36px.

**recreateContainer function:**
```typescript
export async function recreateContainer(
  name: string,
  input: ContainerCreateInput,
): Promise<{success: boolean; message: string; containerId: string}>
```

Implementation steps:
1. Check if the container name matches PROTECTED_CONTAINER_PATTERNS -- if so, throw `new Error('[protected-container] Cannot recreate protected container: ${name}')`. Use the same `isProtected` logic from `manageContainer` (check if any pattern is included in the lowercase name).
2. Get the existing container via `docker.getContainer(name)`.
3. Inspect it to verify it exists (catch 404 and throw `[not-found]` error).
4. Stop the container if it's running: `await container.stop().catch(() => {})` (ignore errors if already stopped).
5. Remove the container: `await container.remove({force: true})`.
6. Override `input.name` with the original `name` to ensure the recreated container keeps the same name.
7. Call the existing `createContainer(input)` function to create and start the new container.
8. Return the result from createContainer.

Wrap in try/catch: if err.statusCode === 404, throw `[not-found]` error. Re-throw other errors.

**renameContainer function:**
```typescript
export async function renameContainer(
  name: string,
  newName: string,
): Promise<{success: boolean; message: string}>
```

Implementation steps:
1. Check if the container name matches PROTECTED_CONTAINER_PATTERNS -- if so, throw `[protected-container]` error.
2. Validate newName matches the Docker naming regex: `/^[a-zA-Z0-9][a-zA-Z0-9_.-]*$/`. If not, throw an error with a descriptive message.
3. Get the container via `docker.getContainer(name)`.
4. Call `await container.rename({name: newName})`.
5. Return `{success: true, message: "Container '${name}' renamed to '${newName}'"}`.
6. Catch 404 and throw `[not-found]` error. Catch 409 (conflict) and throw a descriptive `[conflict]` error about name already in use.

Both functions use the `docker` singleton (the Dockerode instance already imported at the top of docker.ts).

2. Add `recreateContainer` tRPC mutation after the existing `createContainer` mutation. Use the exact same Zod schema from createContainer for the `config` field, plus a `name` field for the target container:

```typescript
recreateContainer: adminProcedure
  .input(z.object({
    name: z.string().min(1).max(255),
    config: z.object({
      // EXACT same fields as createContainer's input schema
      name: z.string().min(1).max(255).regex(/^[a-zA-Z0-9][a-zA-Z0-9_.-]*$/),
      image: z.string().min(1).max(500),
      command: z.array(z.string()).optional(),
      entrypoint: z.array(z.string()).optional(),
      workingDir: z.string().max(500).optional(),
      user: z.string().max(100).optional(),
      hostname: z.string().max(255).optional(),
      domainname: z.string().max(255).optional(),
      tty: z.boolean().optional(),
      openStdin: z.boolean().optional(),
      ports: z.array(z.object({
        hostPort: z.number().int().min(1).max(65535),
        containerPort: z.number().int().min(1).max(65535),
        protocol: z.enum(['tcp', 'udp']),
      })).optional(),
      volumes: z.array(z.object({
        hostPath: z.string().optional(),
        containerPath: z.string().min(1),
        readOnly: z.boolean().optional(),
        type: z.enum(['bind', 'volume', 'tmpfs']),
        volumeName: z.string().optional(),
      })).optional(),
      env: z.array(z.object({ key: z.string().min(1), value: z.string() })).optional(),
      labels: z.array(z.object({ key: z.string().min(1), value: z.string() })).optional(),
      restartPolicy: z.object({
        name: z.enum(['no', 'always', 'on-failure', 'unless-stopped']),
        maximumRetryCount: z.number().int().min(0).optional(),
      }).optional(),
      resources: z.object({
        memoryLimit: z.number().int().min(0).optional(),
        cpuLimit: z.number().int().min(0).optional(),
        cpuShares: z.number().int().min(0).optional(),
      }).optional(),
      healthCheck: z.object({
        test: z.array(z.string()).optional(),
        interval: z.number().int().min(0).optional(),
        timeout: z.number().int().min(0).optional(),
        retries: z.number().int().min(0).optional(),
        startPeriod: z.number().int().min(0).optional(),
      }).optional(),
      networkMode: z.string().max(255).optional(),
      dns: z.array(z.string()).optional(),
      extraHosts: z.array(z.string()).optional(),
      pullImage: z.boolean().optional(),
      autoStart: z.boolean().optional(),
    }),
  }))
  .mutation(async ({input}) => {
    try {
      return await recreateContainer(input.name, input.config)
    } catch (err: any) {
      if (err.message?.includes('[protected-container]')) {
        throw new TRPCError({ code: 'FORBIDDEN', message: err.message.replace('[protected-container] ', '') })
      }
      if (err.message?.includes('[not-found]')) {
        throw new TRPCError({ code: 'NOT_FOUND', message: err.message.replace('[not-found] ', '') })
      }
      if (err.message?.includes('[image-not-found]')) {
        throw new TRPCError({ code: 'NOT_FOUND', message: err.message.replace('[image-not-found] ', '') })
      }
      throw new TRPCError({ code: 'INTERNAL_SERVER_ERROR', message: err.message || `Failed to recreate container ${input.name}` })
    }
  }),
```

3. Add `getDockerEvents` function at the bottom of docker.ts:
   - Signature: `export async function getDockerEvents(options: {since?: number; until?: number; filters?: {type?: string[]}}): Promise<DockerEvent[]>`
   - Compute `since` default: if not provided, use `Math.floor(Date.now() / 1000) - 3600` (last 1 hour)
   - Compute `until` default: if not provided, use `Math.floor(Date.now() / 1000)`
   - Build Docker API query object: `{since: sinceTs, until: untilTs, filters: JSON.stringify(filters.type ? {type: filters.type} : {})}`
   - Call `docker.getEvents(query)` which returns a ReadableStream
   - Collect stream data into a string buffer, then split by newlines, JSON.parse each non-empty line
   - Map each raw event to DockerEvent: `{type: raw.Type, action: raw.Action, actor: raw.Actor?.Attributes?.name || raw.Actor?.Attributes?.image || raw.Actor?.ID?.slice(0, 12) || 'unknown', actorId: raw.Actor?.ID || '', time: raw.time || raw.timeNano ? Math.floor(raw.timeNano / 1e9) : 0, attributes: raw.Actor?.Attributes || {}}`
   - Return the last 200 events sorted by time descending (most recent first)
   - Wrap stream reading in a Promise with a 10-second timeout (setTimeout + stream.destroy) to prevent hanging if Docker streams indefinitely
   - IMPORTANT: dockerode `getEvents()` with both `since` AND `until` returns a finite stream (historical events between those timestamps), NOT an infinite real-time stream. The stream will end naturally when all events in the range are delivered.

2. Add `dockerEvents` query route after the stack management section (at the end of the router, before the closing `})`):

```typescript
// -----------------------------------------------------------------------
// Docker Events (Phase 46)
// -----------------------------------------------------------------------

dockerEvents: adminProcedure
  .input(
    z.object({
      since: z.number().optional(),
      until: z.number().optional(),
      filters: z.object({
        type: z.array(z.string()).optional(),
      }).optional(),
    }).optional(),
  )
  .query(async ({input}) => {
    try {
      return await getDockerEvents({
        since: input?.since,
        until: input?.until,
        filters: input?.filters,
      })
    } catch (err: any) {
      throw new TRPCError({
        code: 'INTERNAL_SERVER_ERROR',
        message: err.message || 'Failed to get Docker events',
      })
    }
  }),
```

  }),

Create `agents-panel.tsx` as a new file in the ai-chat directory. This is a self-contained component with list/detail navigation (per AGNT-02 and AGNT-03).

**Component structure:**

```
AgentsPanel (default export)
  |-- AgentList (list mode)
  |     |-- Agent cards with status badge, last run time, run count
  |     |-- Empty state with IconRobot
  |-- AgentDetail (detail mode)
        |-- Header with back button + agent name
        |-- Status + tier badges
        |-- Last result section
        |-- Configuration section (description, tools, schedule, system prompt)
        |-- Chat history section
```

**Key implementation details:**

1. **Navigation state:** Use discriminated union `type AgentsView = { mode: 'list' } | { mode: 'detail'; agentId: string }` with `useState<AgentsView>({ mode: 'list' })`.

2. **Agent list (AGNT-02):**
   - Use `trpcReact.ai.listSubagents.useQuery(undefined, { refetchInterval: 5_000 })` for 5-second polling
   - Each agent card shows: name, description (truncated to 1 line), status badge, last run time (using `formatDistanceToNow` from date-fns), run count, tier badge
   - Status badges: green for `'active'` (`bg-green-500/20 text-green-400`), yellow for `'paused'` (`bg-yellow-500/20 text-yellow-400`), red for `'stopped'` (`bg-red-500/20 text-red-400`)
   - Tier badge: small text showing `flash`/`sonnet`/`opus` in muted style (`text-text-tertiary text-[10px]`)
   - Cards are clickable, calling `setView({ mode: 'detail', agentId: agent.id })`
   - Loading state: `<IconLoader2 size={24} className='animate-spin text-text-tertiary' />` centered
   - Empty state: `<IconRobot size={40} className='text-text-tertiary' />` with "No agents yet" text and a brief hint "Create agents from the Agents page or let AI create them autonomously"

3. **Agent detail (AGNT-03):**
   - Use `trpcReact.ai.getSubagent.useQuery({ id: agentId })` for full config
   - Use `trpcReact.ai.getSubagentHistory.useQuery({ id: agentId, limit: 50 })` for chat history
   - **Header:** Back button (`<IconArrowLeft size={16} />`) + agent name as title. Clicking back calls `onBack()` which sets view back to list.
   - **Status section:** Status badge (same colors as list) + tier badge + run count + last run time
   - **Last result:** If `lastResult` exists, show in a `bg-surface-1 rounded-radius-sm p-3` box with `text-caption`. Truncate to ~200 chars with "..." if longer.
   - **Configuration section:** Collapsible or always-visible section showing:
     - Description (full text)
     - Tools list (comma-separated or badges)
     - Schedule (cron expression if set, otherwise "No schedule")
     - System prompt (if set, show first ~100 chars in monospace)
   - **Chat history:** Render `SubagentMessage[]` as simple chat bubbles -- user messages right-aligned with `bg-brand/10`, assistant messages left-aligned with `bg-surface-1`. Show `formatDistanceToNow(msg.ts)` timestamp. If empty, show "No conversation history".
   - Loading state while detail data loads: centered spinner

4. **Imports needed:**
   ```typescript
   import {useState} from 'react'
   import {IconRobot, IconArrowLeft, IconLoader2, IconClock, IconPlayerPlay} from '@tabler/icons-react'
   import {formatDistanceToNow} from 'date-fns'
   import {trpcReact} from '@/trpc/trpc'
   import {cn} from '@/shadcn-lib/utils'
   ```

5. **Styling:** Use the project's existing Tailwind classes. The panel fills `flex-1 overflow-hidden` from the parent in index.tsx, so the root element should be `<div className='flex h-full flex-col'>` with the content area as `<div className='flex-1 overflow-y-auto p-3'>`.

6. **Read-only:** This panel is strictly read-only per Phase 21 scope. No start/stop/send controls (deferred to Phase 22).

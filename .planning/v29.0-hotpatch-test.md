7. Phase 33 trap `phase33_finalize` fires on update.sh's clean exit
8. Trap writes `<ts>-success.json` with `from_sha: 7172415b`, `to_sha: <new>`
9. Trap renames `update-<ts>-pending.log` → `update-<ts>-<7sha>.log`
10. New livinityd boots, reads `update-history/`, Past Deploys table renders 4th row

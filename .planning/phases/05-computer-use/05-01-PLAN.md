2. Create `agent/src/tools/mouse.ts` with these 6 exported functions. All functions share a single lazy-load guard for `@jitsi/robotjs` using synchronous `require()` (NOT async import -- robotjs uses CJS via node-gyp-build):

```typescript
import type * as RobotJS from '@jitsi/robotjs';

let robot: typeof RobotJS | null = null;
let loadError: string | null = null;
let loaded = false;

function ensureRobotLoaded(): void {
  if (loaded) return;
  loaded = true;
  try {
    robot = require('@jitsi/robotjs');
  } catch (err: unknown) {
    loadError = err instanceof Error ? err.message : String(err);
  }
}
```

The standard return type for ALL tool functions is:
```typescript
Promise<{ success: boolean; output: string; error?: string; data?: unknown }>
```

**executeMouseClick(params)** -- MOUSE-01:
- Extract `x: number`, `y: number` from params. Validate both are numbers >= 0, return error if not.
- Call `robot!.moveMouse(x, y)` then `robot!.mouseClick('left')`.
- Return `{ success: true, output: 'Left-clicked at (x, y)', data: { x, y, button: 'left' } }`.

**executeMouseDoubleClick(params)** -- MOUSE-02:
- Same params as click. Validate x, y.
- Call `robot!.moveMouse(x, y)` then `robot!.mouseClick('left', true)` (second param = double-click).
- Return with `button: 'double'` in data.

**executeMouseRightClick(params)** -- MOUSE-03:
- Same params as click. Validate x, y.
- Call `robot!.moveMouse(x, y)` then `robot!.mouseClick('right')`.
- Return with `button: 'right'` in data.

**executeMouseMove(params)** -- MOUSE-04:
- Extract `x: number`, `y: number`. Validate >= 0.
- Call `robot!.moveMouse(x, y)` (instant, NOT moveMouseSmooth).
- Return `{ success: true, output: 'Moved mouse to (x, y)', data: { x, y } }`.

**executeMouseDrag(params)** -- MOUSE-05:
- Extract `fromX`, `fromY`, `toX`, `toY` as numbers. Validate all 4.
- Sequence: `moveMouse(fromX, fromY)` -> `mouseToggle('down')` -> try { `moveMouse(toX, toY)` } finally { `mouseToggle('up')` }.
- The try/finally is CRITICAL to prevent stuck mouse buttons on error.
- Return `{ success: true, output: 'Dragged from (...) to (...)', data: { fromX, fromY, toX, toY } }`.

**executeMouseScroll(params)** -- MOUSE-06:
- Extract `direction: string` (required, "up" or "down"), `amount: number` (optional, default 3), `x: number` (optional), `y: number` (optional).
- If x and y are provided (both numbers), call `robot!.moveMouse(x, y)` first.
- Calculate scrollY: direction === 'up' ? amount : -amount.
- Call `robot!.scrollMouse(0, scrollY)`.
- Return success with direction, amount, and optional x/y in data.

3. Create `agent/src/tools/keyboard.ts` with these 2 exported functions. It shares the same lazy-load pattern (same `ensureRobotLoaded`, same `robot` variable -- duplicate the lazy-load block in this file too, don't try to share across files):

**executeKeyboardType(params)** -- KEY-01:
- Extract `text: string`. Validate it is a non-empty string.
- Call `robot!.typeString(text)`.
- Return `{ success: true, output: 'Typed N characters', data: { length: text.length } }`.

**executeKeyboardPress(params)** -- KEY-02:
- Extract `key: string`. Validate non-empty string.
- Build a KEY_ALIASES normalization map: `ctrl` -> `control`, `ctl` -> `control`, `cmd` -> `command`, `meta` -> `command`, `win` -> `command`, `super` -> `command`, `esc` -> `escape`, `return` -> `enter`, `cr` -> `enter`, `del` -> `delete`, `ins` -> `insert`, `pgup` -> `pageup`, `pgdn` -> `pagedown`, `bs` -> `backspace`.
- Define MODIFIER_KEYS set: `['control', 'alt', 'shift', 'command']`.
- Parse: split key string by `+`, normalize each part via KEY_ALIASES (toLowerCase + lookup), separate into modifiers and mainKey.
- If only modifiers and no mainKey, pop the last modifier as mainKey.
- Call: if modifiers.length > 0 then `robot!.keyTap(mainKey, modifiers)`, else `robot!.keyTap(mainKey)`.
- Build combo string for output: `modifiers.join('+') + '+' + mainKey` or just `mainKey`.
- Return `{ success: true, output: 'Pressed key: combo', data: { key: mainKey, modifiers } }`.

**Install prerequisites on Server5** (run these commands via SSH):

1. **Node.js 22 LTS** (via nvm if not installed, or NodeSource):
   ```bash
   curl -fsSL https://deb.nodesource.com/setup_22.x | bash -
   apt-get install -y nodejs
   node --version  # should be 22.x
   ```

2. **PostgreSQL 16**:
   ```bash
   apt-get install -y postgresql-16 postgresql-client-16
   systemctl enable postgresql
   systemctl start postgresql
   # Create platform database and user:
   sudo -u postgres psql -c "CREATE USER platform WITH PASSWORD 'LivPlatform2024!';"
   sudo -u postgres psql -c "CREATE DATABASE platform OWNER platform;"
   sudo -u postgres psql -c "GRANT ALL PRIVILEGES ON DATABASE platform TO platform;"
   ```
   If postgresql-16 is not available via apt, use the PostgreSQL apt repository first:
   ```bash
   echo "deb http://apt.postgresql.org/pub/repos/apt $(lsb_release -cs)-pgdg main" > /etc/apt/sources.list.d/pgdg.list
   wget --quiet -O - https://www.postgresql.org/media/keys/ACCC4CF8.asc | apt-key add -
   apt-get update
   apt-get install -y postgresql-16
   ```

3. **Redis 7**:
   ```bash
   apt-get install -y redis-server
   # Set password and memory limits:
   sed -i 's/# requirepass foobared/requirepass LivRelayRedis2024!/' /etc/redis/redis.conf
   sed -i 's/# maxmemory <bytes>/maxmemory 512mb/' /etc/redis/redis.conf
   sed -i 's/# maxmemory-policy noeviction/maxmemory-policy allkeys-lru/' /etc/redis/redis.conf
   systemctl restart redis-server
   systemctl enable redis-server
   # Verify:
   redis-cli -a 'LivRelayRedis2024!' ping  # should return PONG
   ```

4. **Caddy** (bare install -- no configuration, Phase 10 handles Caddy config):
   ```bash
   apt-get install -y debian-keyring debian-archive-keyring apt-transport-https curl
   curl -1sLf 'https://dl.cloudsmith.io/public/caddy/stable/gpg.key' | gpg --dearmor -o /usr/share/keyrings/caddy-stable-archive-keyring.gpg
   curl -1sLf 'https://dl.cloudsmith.io/public/caddy/stable/debian.deb.txt' | tee /etc/apt/sources.list.d/caddy-stable.list
   apt-get update
   apt-get install -y caddy
   caddy version  # verify installation
   systemctl stop caddy  # stop default service -- Phase 10 configures and starts it
   systemctl disable caddy  # don't auto-start until Phase 10 configures it
   ```

5. **PM2**:

   npm install -g pm2 tsx

Structure:
```typescript
import {useIsMobile} from '@/hooks/use-is-mobile'
import {useMobileApp} from './mobile-app-context'
import {IconHome2, IconMessageCircle, IconFolder, IconSettings, IconServer} from '@tabler/icons-react'

  {id: 'ai-chat', label: 'AI Chat', icon: IconMessageCircle, appId: 'LIVINITY_ai-chat', route: '/ai-chat', appIcon: '/figma-exports/dock-home.png'},
  {id: 'files', label: 'Files', icon: IconFolder, appId: 'LIVINITY_files', route: '/files/Home', appIcon: '/figma-exports/dock-files.png'},
  {id: 'settings', label: 'Settings', icon: IconSettings, appId: 'LIVINITY_settings', route: '/settings', appIcon: '/figma-exports/dock-settings.png'},
  {id: 'server', label: 'Server', icon: IconServer, appId: 'LIVINITY_server-control', route: '/server-control', appIcon: '/figma-exports/dock-live-usage.png'},
] as const
```

The component:
- Returns null if `!isMobile` (desktop unchanged)
- Fixed position at bottom: `fixed bottom-0 left-0 right-0 z-[60]` (z-60 to sit above MobileAppRenderer's z-50)
- Background: `bg-white/95 backdrop-blur-md border-t border-black/10` (iOS-style solid white with subtle top border)
- Safe area bottom: `pb-safe` (from Phase 37 tailwindcss-safe-area plugin)
- Height: 56px content area (the bar itself, excluding safe area padding)
- Layout: flex row, 5 equal-width tabs, each a button
- Each tab button: flex-col, items-center, justify-center, gap-0.5
- Icon: 24x24 (size={24} stroke={1.5})
- Label: text-[10px] font-medium
- Active state: determined by comparing `activeApp?.appId` to tab's `appId`. Home tab is active when `activeApp === null`.
- Active color: `text-blue-500` (brand). Inactive: `text-gray-400`
- onClick behavior:
  - Home tab: call `closeApp()` (returns to grid)
  - Other tabs: call `openApp(tab.appId, tab.route, tab.label, tab.appIcon)` -- but if the tab is already active (`activeApp?.appId === tab.appId`), do nothing (prevent re-open)

Export: `export function MobileTabBar()`

Do NOT add any animations (deferred per user decision context).

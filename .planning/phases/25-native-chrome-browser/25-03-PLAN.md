**use-launch-app.ts — Call nativeStart before opening native app URL:**

1. Add `trpcClient` import at the top: `import {trpcClient} from '@/trpc/trpc'` (the imperative client, not the React hooks — needed because the launch function is not a React hook itself and must fire-and-forget an async mutation).

2. Add a `nativeStart` mutation via React hooks at the top of the hook body (alongside `trackOpen`):

3. Change the returned function from a synchronous function to an async function. Replace:
```typescript

```
with:

4. Before the `openInTab(opts?.path)` call at the end, add a native app start block. Insert BEFORE the final `openInTab` call:

```typescript
// Auto-start native app stream before opening URL
if (app.native) {
  try {
    await nativeStartMut.mutateAsync({appId})
```

The full flow becomes: credential check -> tor check -> if native: await nativeStart -> openInTab.

**setup-chrome-native.sh — Add --remote-debugging-port=9222 to Chrome launch (lines 53-62):**

In the ExecStart section of the systemd service unit, add `--remote-debugging-port=9222` and `--remote-debugging-address=127.0.0.1` to the google-chrome-stable command. The updated Chrome launch becomes:

```bash
ExecStart=/bin/bash -c '\
  sleep 1 && \
  google-chrome-stable \
    --no-sandbox \
    --disable-gpu \
    --disable-dev-shm-usage \
    --no-first-run \
    --start-maximized \
    --remote-debugging-port=9222 \
    --remote-debugging-address=127.0.0.1 \
    --user-data-dir=${CHROME_DATA_DIR} \
    --display=:99 & \
  sleep 2 && \
  x11vnc -display :99 -nopw -listen 127.0.0.1 -xkb -ncache 10 -forever -bg -rfbport 5900 && \
  websockify --daemon --web /opt/noVNC 6080 127.0.0.1:5900'
```

This enables Chrome DevTools Protocol on port 9222 (localhost only for security) so that the MCP Playwright server can control Chrome.

NOTE: The setup script change means `setup-chrome-native.sh` must be re-run on Server4 (or the service file manually updated) for CDP to be available. This is a server-side deployment step that should be noted in the SUMMARY.

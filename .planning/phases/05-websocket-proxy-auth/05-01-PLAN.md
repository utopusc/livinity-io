Insert a new `/ws/desktop` handler block in the `this.server?.on('upgrade', ...)` handler, AFTER the voice proxy block (line 539) and BEFORE the generic router check (line 542: `const wss = this.webSocketRouter.get(pathname)`).

The handler follows the exact same structure as the `/ws/voice` proxy (lines 476-539) but with these critical differences:

1. **TCP upstream instead of WebSocket upstream**: Use `createConnection({ host: '127.0.0.1', port: 5900 })` instead of `new WebSocket(...)`. This is because VNC uses raw TCP (RFB protocol), not WebSocket.

2. **Origin header validation**: Before JWT check, validate the Origin header. Read the domain config from Redis (`livos:domain:config`). If Origin header is present, it must match `https://{domain}` or `https://pc.{domain}` or `https://*.{domain}`. If it does not match, write `HTTP/1.1 403 Forbidden\r\n\r\n` and destroy socket. If no Origin header (non-browser client), allow the connection (VNC clients may not send Origin).

3. **JWT auth**: Extract token from `searchParams.get('token')`. If no token, also check for LIVINITY_SESSION cookie in request headers (fallback for subdomain access where cookie is set by Caddy). Verify with `this.verifyToken(token)`. On failure, write `HTTP/1.1 401 Unauthorized\r\n\r\n` and destroy socket.

4. **NativeApp auto-start**: After auth succeeds, get the desktop-stream NativeApp via `this.livinityd.apps.getNativeApp('desktop-stream')`. If it doesn't exist or has no GUI (`livos:desktop:has_gui` Redis key is not 'true'), write `HTTP/1.1 503 Service Unavailable\r\n\r\n` with body and destroy socket. If app state is not 'ready', call `await desktopApp.start()` and handle failure with 503.

5. **Idle timer reset**: Call `desktopApp.resetIdleTimer()` on each new connection.

6. **Bidirectional binary relay**: After TCP connects:
   - `ws.on('message', (data) => { if (vnc.writable) vnc.write(Buffer.from(data as ArrayBuffer)) })`
   - `vnc.on('data', (data) => { if (ws.readyState === 1) ws.send(data) })`

7. **Cleanup on disconnect**:
   - `ws.on('close', () => vnc.destroy())` -- destroys TCP connection only, NOT the x11vnc service
   - `vnc.on('close', () => ws.close())`
   - `vnc.on('error', (err) => { this.logger.error('VNC TCP error', err); ws.close(1011, 'VNC connection error') })`
   - `ws.on('error', () => vnc.destroy())`

8. **WebSocket binary type**: Set `ws.binaryType = 'nodebuffer'` after upgrade to ensure binary frames are handled correctly.

The complete handler structure:

```typescript
// ── Desktop Stream WebSocket-to-TCP Proxy ──────────────────
if (pathname === '/ws/desktop') {
    // 1. Origin validation
    const origin = request.headers.origin
    if (origin) {
        const domainConfigRaw = await this.livinityd.ai.redis.get('livos:domain:config').catch(() => null)
        let allowedDomain = ''
        if (domainConfigRaw) {
            const dc = JSON.parse(domainConfigRaw)
            if (dc.active && dc.domain) allowedDomain = dc.domain
        }
        if (allowedDomain) {
            const originUrl = new URL(origin)
            const originHost = originUrl.hostname
            if (originHost !== allowedDomain && !originHost.endsWith('.' + allowedDomain)) {
                this.logger.verbose('WS desktop rejected: origin mismatch', origin)
                socket.write('HTTP/1.1 403 Forbidden\r\n\r\n')
                socket.destroy()
                return
            }
        }
    }

    // 2. JWT auth (token from query param or LIVINITY_SESSION cookie)
    let token = searchParams.get('token')
    if (!token) {
        const cookieHeader = request.headers.cookie || ''
        const sessionMatch = cookieHeader.match(/LIVINITY_SESSION=([^;]+)/)
        if (sessionMatch) token = sessionMatch[1]
    }
    if (!token) {
        this.logger.verbose('WS desktop rejected: no token')
        socket.write('HTTP/1.1 401 Unauthorized\r\n\r\n')
        socket.destroy()
        return
    }
    const isValid = await this.verifyToken(token).catch(() => false)
    if (!isValid) {
        this.logger.verbose('WS desktop rejected: invalid token')
        socket.write('HTTP/1.1 401 Unauthorized\r\n\r\n')
        socket.destroy()
        return
    }

    // 3. NativeApp auto-start
    const hasGui = await this.livinityd.ai.redis.get('livos:desktop:has_gui').catch(() => null)
    if (hasGui !== 'true') {
        this.logger.verbose('WS desktop rejected: no GUI available')
        socket.write('HTTP/1.1 503 Service Unavailable\r\n\r\n')
        socket.destroy()
        return
    }
    const desktopApp = this.livinityd.apps.getNativeApp('desktop-stream')
    if (!desktopApp) {
        this.logger.verbose('WS desktop rejected: desktop-stream app not registered')
        socket.write('HTTP/1.1 503 Service Unavailable\r\n\r\n')
        socket.destroy()
        return
    }
    if (desktopApp.state !== 'ready') {
        try {
            await desktopApp.start()
        } catch (err) {
            this.logger.error('WS desktop: failed to start x11vnc', err)
            socket.write('HTTP/1.1 503 Service Unavailable\r\n\r\n')
            socket.destroy()
            return
        }
    }
    desktopApp.resetIdleTimer()

    // 4. Create TCP connection to x11vnc
    const vnc = createConnection({ host: '127.0.0.1', port: 5900 })
    const desktopWss = new WebSocketServer({ noServer: true })

    vnc.on('connect', () => {
        this.logger.verbose('WS desktop: VNC TCP connected, upgrading client')
        desktopWss.handleUpgrade(request, socket, head, (ws) => {
            desktopWss.close()
            ws.binaryType = 'nodebuffer'

            // Bidirectional binary relay
            ws.on('message', (data) => {
                if (vnc.writable) vnc.write(Buffer.from(data as ArrayBuffer))
            })
            vnc.on('data', (data) => {
                if (ws.readyState === 1) ws.send(data)
            })

            // Cleanup: close one side when the other disconnects
            ws.on('close', () => vnc.destroy())
            vnc.on('close', () => ws.close())
            vnc.on('error', (err) => {
                this.logger.error('WS desktop: VNC TCP error', err)
                ws.close(1011, 'VNC connection error')
            })
            ws.on('error', () => vnc.destroy())
        })
    })

    vnc.on('error', (err) => {
        this.logger.error('WS desktop: VNC TCP connect error', err)
        desktopWss.close()
        socket.write('HTTP/1.1 502 Bad Gateway\r\n\r\n')
        socket.destroy()
    })

    return
}
```

IMPORTANT: The handler MUST NOT call `desktopApp.stop()` when the WebSocket closes. Only `vnc.destroy()` is called, which closes the TCP socket to x11vnc. The x11vnc process itself continues running (it has `-forever` flag) and the NativeApp idle timer handles eventual shutdown after 30 minutes of no connections.

let lastIdleReset = Date.now()
vnc.on('data', (data) => {
    if (ws.readyState === 1) ws.send(data)
    // Throttled idle timer reset on data activity (max once per 60s)

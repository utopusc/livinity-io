Component `ClockWidget` with props `{config?: Record<string, unknown>}`:

- Read mode from `config?.mode` defaulting to `'digital'`
- Use `useState` + `useEffect` with a 1-second `setInterval` to update current time
- Store `new Date()` in state, update every 1000ms, cleanup interval on unmount

**Digital mode** (default):
- Large time display: `text-3xl font-light text-white/90 tabular-nums` showing `HH:MM` (24h format using `toLocaleTimeString('tr-TR', {hour: '2-digit', minute: '2-digit'})`)
- Seconds in smaller font: `text-lg text-white/50 tabular-nums` showing `:SS`
- Date below: `text-xs text-white/60` showing day name + date (e.g. "Tuesday, 18 March") using `toLocaleDateString('en-US', {weekday: 'long', day: 'numeric', month: 'long'})`
- Layout: centered vertically and horizontally in the container with `items-center justify-center p-4`

**Analog mode** (when `config?.mode === 'analog'`):
- SVG clock face, 120x120 viewBox
- Circle with `stroke="rgba(255,255,255,0.3)"` and `fill="none"` for the face
- Hour markers: 12 small circles at clock positions (use `Math.cos`/`Math.sin` with angle = `i * 30 * Math.PI / 180 - Math.PI/2`)
- Three hands (hour, minute, second) as `<line>` elements with `stroke="rgba(255,255,255,0.9)"`:
  - Hour hand: `strokeWidth={3}`, length 30px, angle = `(hours % 12 + minutes / 60) * 30`
  - Minute hand: `strokeWidth={2}`, length 40px, angle = `minutes * 6`
  - Second hand: `strokeWidth={1}`, length 45px, angle = `seconds * 6`, color `rgba(255,255,255,0.5)`
- Center dot: small filled circle
- Date text below the SVG: `text-[10px] text-white/60`
- All angles measured from 12 o'clock position (subtract 90 degrees)
- Use CSS `transition: transform 0.3s ease` on the second hand for smooth movement

Export as named export: `export function ClockWidget({config}: {config?: Record<string, unknown>})`

Component `QuickNotesWidget` with props `{widgetId: string}`:
- The preference key is `widget-notes-${widgetId}` — unique per widget instance
- Load initial value from `trpcReact.preferences.get.useQuery({keys: [`widget-notes-${widgetId}`]})`
- Use `useState<string>` for the text content, initialized to `''`
- Use `useRef` to track if server data has been synced (same `serverSynced` pattern)
- On server data load, set text if remote value exists
- Also load from `localStorage.getItem(`widget-notes-${widgetId}`)` as initial value
- **Auto-save with debounce**: Use `useRef` for a debounce timer. On every text change:
  1. Save to localStorage immediately: `localStorage.setItem(`widget-notes-${widgetId}`, text)`
  2. Clear previous timer, set new `setTimeout` of 1000ms
  3. On timeout, call `setPref.mutate({key: `widget-notes-${widgetId}`, value: text})`
- Layout: `WidgetContainer` with full height
  - Header: `px-3 pt-2 pb-1` with `text-[11px] text-white/50 font-medium` showing "Notes"
  - Textarea: `flex-1 w-full resize-none bg-transparent px-3 pb-3 text-sm text-white/90 placeholder:text-white/30 focus:outline-none`
    - `placeholder="Type something..."`
    - `value={text}`
    - `onChange={e => setText(e.target.value)}`
    - `onPointerDown={e => e.stopPropagation()}` — CRITICAL: stops DnD from activating when clicking inside the textarea
    - `onMouseDown={e => e.stopPropagation()}` — same reason, backup for mouse events
  - The textarea MUST call `e.stopPropagation()` on pointer/mouse down to prevent `@dnd-kit` from starting a drag when the user is trying to type or select text

**File 3: `widget-renderer.tsx`** — Maps WidgetMeta to component:

```typescript
import {WidgetMeta} from './widget-types'
import {ClockWidget} from './clock-widget'
import {SystemInfoCompactWidget} from './system-info-compact-widget'
import {SystemInfoDetailedWidget} from './system-info-detailed-widget'
import {QuickNotesWidget} from './quick-notes-widget'

export function WidgetRenderer({widget}: {widget: WidgetMeta}) {
  switch (widget.type) {
    case 'clock':
      return <ClockWidget config={widget.config} />
    case 'system-info-compact':
      return <SystemInfoCompactWidget />
    case 'system-info-detailed':
      return <SystemInfoDetailedWidget />
    case 'quick-notes':
      return <QuickNotesWidget widgetId={widget.id} />
    default:
      return (
        <div className="flex h-full w-full items-center justify-center rounded-2xl border border-white/20 bg-white/10 text-white/40 text-xs backdrop-blur-xl">
          Unknown widget
        </div>
      )
  }
}
```

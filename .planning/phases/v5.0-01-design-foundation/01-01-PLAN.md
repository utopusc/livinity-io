## Objective

Convert all semantic color tokens and box shadows in `tailwind.config.ts` from dark theme (white-on-black) to light theme (dark-on-white). This is the single most impactful change — every component using semantic tokens will immediately render in light mode.

## must_haves
- All surface-* tokens use black-based opacity for light backgrounds
- All text-* tokens use dark slate-based colors
- All border-* tokens use black-based opacity
- All elevation shadows use subtle black drop shadows without white inner glows
- Component shadows (dock, dialog, dropdown, etc.) produce natural light-mode elevation
- Typography prose uses dark-on-light values

Add five Claude auth routes in api.ts, immediately after the Kimi logout route (after line ~387). Follow the exact Kimi auth pattern from lines 198-387. Access ClaudeProvider via `brain.getProviderManager().getProvider('claude')`.

Add this import at the top of api.ts (alongside existing imports):
```typescript
import type { ClaudeProvider } from './providers/claude.js';
```

Add these routes inside the createApiServer function, after the Kimi logout route block:

1. **POST /api/claude/set-api-key** (per AUTH-01, AUTH-02):
   - Accepts `{ apiKey: string }` in request body
   - Validates apiKey is a non-empty string starting with `sk-ant-`
   - Validates against Anthropic API by calling `GET https://api.anthropic.com/v1/models` with headers `x-api-key: {apiKey}`, `anthropic-version: 2023-06-01`
   - On success: stores key in Redis via `redis.set('nexus:config:anthropic_api_key', apiKey)`, sets auth method via `redis.set('nexus:config:claude_auth_method', 'api-key')`, responds `{ success: true }`
   - On validation failure: responds 400 `{ error: 'Invalid API key format' }`
   - On API validation failure: responds 401 `{ error: 'API key validation failed: {status}' }`

2. **GET /api/claude/status** (per AUTH-01):
   - Gets ClaudeProvider via `brain.getProviderManager().getProvider('claude') as ClaudeProvider | undefined`
   - If provider not found: respond 503 `{ error: 'Claude provider not available' }`
   - Reads auth method from Redis: `redis.get('nexus:config:claude_auth_method')` (default: 'api-key')
   - If auth method is 'api-key': check if key exists via `redis.get('nexus:config:anthropic_api_key')`, return `{ authenticated: !!key, method: 'api-key', provider: 'claude' }`
   - If auth method is 'sdk-subscription': call `provider.getCliStatus()`, return `{ authenticated: status.authenticated, method: 'sdk-subscription', provider: 'claude', user: status.user, cliInstalled: status.installed }`

3. **POST /api/claude/start-login** (per AUTH-03):
   - Gets ClaudeProvider, calls `provider.startLogin()`
   - Sets auth method: `redis.set('nexus:config:claude_auth_method', 'sdk-subscription')`
   - Returns the result from startLogin() directly: `{ url, error, alreadyAuthenticated }`

4. **POST /api/claude/submit-code** (per AUTH-03):
   - Accepts `{ code: string }` in request body
   - Gets ClaudeProvider, calls `provider.submitLoginCode(code)`
   - Returns the result directly: `{ success, error }`

5. **POST /api/claude/logout**:
   - Gets ClaudeProvider, calls `provider.logout()`
   - Clears Redis keys: `redis.del('nexus:config:anthropic_api_key')`, `redis.del('nexus:config:claude_auth_method')`
   - Returns `{ success: true }`

All routes wrap in try/catch returning 500 with `{ error: formatErrorMessage(err) }` on failure, matching the Kimi pattern.

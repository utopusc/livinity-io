**2. Create agent/build-sea.mjs** -- a complete build script that:
  a. Runs the esbuild bundle (import and call the existing esbuild.config.mjs logic -- or inline it)
  b. Generates the SEA blob: `node --experimental-sea-config sea-config.json`
  c. Copies the Node.js binary: `cp process.execPath dist/livinity-agent.exe` (on Windows) or no extension on other platforms
  d. Removes the code signature on Windows: use `npx postject` with `--sentinel-fuse NODE_SEA_FUSE_fce680ab2cc467b6e072b8b5df1996b2`
  e. Injects the blob: `npx postject dist/livinity-agent.exe NODE_SEA_BLOB dist/sea-prep.blob --sentinel-fuse NODE_SEA_FUSE_fce680ab2cc467b6e072b8b5df1996b2`
  f. Copies native dependencies alongside the binary:
     - `node_modules/systray2/tray_windows_release.exe` -> `dist/traybin/tray_windows_release.exe`
     - `node_modules/systray2/index.js` + `node_modules/systray2/package.json` -> `dist/node_modules/systray2/`
     - The entire `node_modules/systray2/traybin/` directory -> `dist/node_modules/systray2/traybin/`
     - `node_modules/node-screenshots/` and `node_modules/@node-rs/node-screenshots-win32-x64-msvc/` -> `dist/node_modules/` (preserving structure)
  g. Copies `dist/setup-ui/` (already handled by esbuild.config.mjs)
  h. Logs each step with console.log

  Use `child_process.execSync` for running node and postject commands. Use `fs.cpSync` for file copies. Detect platform via `process.platform` to use correct binary names and file extensions.

  IMPORTANT: The script must add `postject` as a devDependency in package.json OR use `npx postject@latest`. Use npx to avoid adding a dependency.

  IMPORTANT: systray2 resolves its binary via `path.join(__dirname, 'traybin', binName)`. Since it's externalized from esbuild with `require = createRequire(import.meta.url)`, at runtime in the SEA binary the require will look relative to the SEA binary's location. So we need `dist/node_modules/systray2/` with index.js and traybin/ inside.

**2. Create agent/installer/windows/setup.iss** -- Inno Setup 6 script with:

```iss
[Setup]
AppId={{A1B2C3D4-E5F6-7890-ABCD-EF1234567890}
AppName=Livinity Agent
AppVersion=0.1.0
AppPublisher=Livinity
AppPublisherURL=https://livinity.io
DefaultDirName={autopf}\Livinity Agent
DefaultGroupName=Livinity
OutputDir=..\..\dist\installer
OutputBaseFilename=LivinityAgentSetup
Compression=lzma2
SolidCompression=yes
WizardStyle=modern
PrivilegesRequired=lowest
; No admin required -- installs to user's AppData if no admin
ArchitecturesInstallIn64BitMode=x64compatible
UninstallDisplayName=Livinity Agent
```

[Files] section:
- `Source: "..\..\dist\livinity-agent.exe"; DestDir: "{app}"; Flags: ignoreversion`
- `Source: "..\..\dist\node_modules\systray2\*"; DestDir: "{app}\node_modules\systray2"; Flags: ignoreversion recursesubdirs createallsubdirs`
- `Source: "..\..\dist\node_modules\node-screenshots\*"; DestDir: "{app}\node_modules\node-screenshots"; Flags: ignoreversion recursesubdirs createallsubdirs`
- `Source: "..\..\dist\node_modules\@node-rs\*"; DestDir: "{app}\node_modules\@node-rs"; Flags: ignoreversion recursesubdirs createallsubdirs`
- `Source: "..\..\dist\setup-ui\*"; DestDir: "{app}\setup-ui"; Flags: ignoreversion recursesubdirs createallsubdirs`

[Icons] section:
- Start Menu: `Name: "{group}\Livinity Agent"; Filename: "{app}\livinity-agent.exe"; Parameters: "start"`
- Start Menu uninstall: `Name: "{group}\Uninstall Livinity Agent"; Filename: "{uninstallexe}"`
- Optional Desktop: `Name: "{autodesktop}\Livinity Agent"; Filename: "{app}\livinity-agent.exe"; Parameters: "start"; Tasks: desktopicon`

[Tasks] section:
- `Name: "desktopicon"; Description: "Create a &desktop shortcut"; GroupDescription: "Additional shortcuts:"; Flags: unchecked`

[Registry] section:
- Auto-start: `Root: HKCU; Subkey: "Software\Microsoft\Windows\CurrentVersion\Run"; ValueType: string; ValueName: "LivinityAgent"; ValueData: """{app}\livinity-agent.exe"" start --background"; Flags: uninsdeletevalue`

[Run] section:
- Post-install launch: `Filename: "{app}\livinity-agent.exe"; Parameters: "start"; Description: "Launch Livinity Agent"; Flags: nowait postinstall skipifsilent`

[UninstallRun] section:
- Stop agent before uninstall: `Filename: "{app}\livinity-agent.exe"; Parameters: "stop"; Flags: runhidden waituntilterminated`

[UninstallDelete] section:
- Clean up PID file: `Type: files; Name: "{userappdata}\.livinity\agent.pid"`

[Code] section (Pascal Script):
- Add a custom page asking if user wants to remove credentials (~/.livinity/) during uninstall
- If user confirms, recursively delete `{userprofile}\.livinity\`

**3. Create agent/installer/windows/build-installer.bat**:
```bat
@echo off
echo Building Livinity Agent Windows Installer...
echo.
echo Step 1: Building SEA binary...
cd /d "%~dp0..\.."
call npm run build:sea
if errorlevel 1 (
    echo FAILED: SEA build failed
    exit /b 1
)
echo.
echo Step 2: Compiling installer with Inno Setup...
where iscc >nul 2>&1
if errorlevel 1 (
    echo ERROR: ISCC.exe not found. Install Inno Setup 6 from https://jrsoftware.org/isdl.php
    echo and add its directory to PATH.
    exit /b 1
)
iscc installer\windows\setup.iss
if errorlevel 1 (
    echo FAILED: Inno Setup compilation failed
    exit /b 1
)
echo.
echo SUCCESS: Installer created at dist\installer\LivinityAgentSetup.exe
```

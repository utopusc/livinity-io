**1. Create `agent/src/tools/shell.ts`:**

Export a single async function:
```typescript
export async function executeShell(params: Record<string, unknown>): Promise<ToolResult>
```

Where `ToolResult` is `{ success: boolean; output: string; error?: string; data?: unknown }`.

Implementation details (per CONTEXT.md decisions):
- Extract params: `command` (string, required), `cwd` (string, optional — default to `os.homedir()`), `timeout` (number, optional — default 30000ms)
- Validate: if `command` is missing or empty, return `{ success: false, output: '', error: 'Missing required parameter: command' }`
- Detect shell based on `process.platform`:
  - `win32`: shell = `'powershell.exe'`, shellArgs = `['-NoProfile', '-NonInteractive', '-Command', command]`
  - others: shell = `'/bin/bash'`, shellArgs = `['-c', command]`. If `/bin/bash` does not exist (check with `fs.existsSync`), fallback to `'/bin/sh'`
- Use `child_process.spawn(shell, shellArgs, { cwd, timeout, env: process.env })` -- NOT `exec`
- Collect stdout and stderr into buffers. On process close, build result.
- Wrap spawn in a Promise. Track `startTime = Date.now()` before spawn, compute `duration = Date.now() - startTime` on close.
- Truncate stdout and stderr individually if either exceeds 100 * 1024 bytes (100KB). Append `\n...[truncated, original size: ${originalSize} bytes]` to truncated output.
- The structured data in the result:
  ```typescript
  data: { stdout, stderr, exitCode: code ?? 1, duration }
  ```
- Set `output` to stdout (for display), or stderr if stdout is empty.
- Set `success: true` always (non-zero exit codes are NOT errors per CONTEXT.md — they're valid command results). Only set `success: false` if the spawn itself fails (ENOENT, etc).
- Catch spawn errors (e.g., shell not found): return `{ success: false, output: '', error: err.message }`

**2. Refactor `agent/src/tools.ts`:**

Replace the current `executeToolStub` function with an async `executeTool` function:
```typescript
import { executeShell } from './tools/shell.js';

export async function executeTool(
  tool: string,
  params: Record<string, unknown>,
): Promise<{ success: boolean; output: string; error?: string; data?: unknown }> {
  switch (tool) {
    case 'shell':
      return executeShell(params);
    // files_list, files_read, etc. — stub for now
    default:
      return { success: false, output: '', error: `Tool '${tool}' is not yet implemented` };
  }
}
```

Keep `TOOL_NAMES` and `ToolName` exports unchanged. Remove `executeToolStub`. Export `executeTool` instead.

**3. Update `agent/src/connection-manager.ts`:**

Create `agent/src/tools/files.ts` with five exported async functions. All use `node:fs/promises` and `node:path`. All return the same `ToolResult` shape: `{ success: boolean; output: string; error?: string; data?: unknown }`.

**Shared helper — safe path resolution:**
```typescript
import { homedir } from 'node:os';
import { resolve, relative } from 'node:path';

const MAX_READ_SIZE = 1024 * 1024; // 1MB

function safePath(inputPath: string): string {
  const home = homedir();
  const resolved = resolve(home, inputPath);
  const rel = relative(home, resolved);
  if (rel.startsWith('..') || resolve(home, rel) !== resolved) {
    throw new Error(`Path traversal blocked: '${inputPath}' escapes home directory`);
  }
  return resolved;
}
```

This resolves paths relative to home, and rejects any path that escapes via `..`.

**1. `executeFilesList(params)`:**
- Extract: `path` (string, required — directory to list)
- Call `safePath(path)`, then `fs.readdir(resolved, { withFileTypes: true })`
- For each entry, call `fs.stat(fullPath)` to get size and mtime
- Return data as array:
  ```typescript
  data: entries.map(e => ({
    name: e.name,
    type: e.isDirectory() ? 'directory' : e.isSymbolicLink() ? 'symlink' : 'file',
    size: stat.size,
    modified: stat.mtime.toISOString(),
  }))
  ```
- Set `output` to a formatted table: `name  type  size  modified` one per line
- Sort entries: directories first, then files, alphabetical within each group
- Wrap stat calls in try/catch individually so one broken symlink doesn't fail the entire listing

**2. `executeFilesRead(params)`:**
- Extract: `path` (string, required), `encoding` (string, optional — default `'utf-8'`)
- Call `safePath(path)`, then `fs.stat(resolved)` to check size
- If `stat.size > MAX_READ_SIZE` (1MB), return `{ success: false, output: '', error: 'File too large (${stat.size} bytes). Maximum: 1MB' }`
- Read with `fs.readFile(resolved, encoding)`
- Return `{ success: true, output: content, data: { content, size: stat.size, path: resolved } }`

**3. `executeFilesWrite(params)`:**
- Extract: `path` (string, required), `content` (string, required)
- Call `safePath(path)`
- Create parent directory if it doesn't exist: `fs.mkdir(dirname(resolved), { recursive: true })`
- Write with `fs.writeFile(resolved, content, 'utf-8')`
- Return `{ success: true, output: 'Written ${Buffer.byteLength(content)} bytes to ${resolved}', data: { bytesWritten: Buffer.byteLength(content), path: resolved } }`

**4. `executeFilesDelete(params)`:**
- Extract: `path` (string, required)
- Call `safePath(path)`, then `fs.rm(resolved)` (NOT recursive — single file only, per CONTEXT.md scope)
- Return `{ success: true, output: 'Deleted ${resolved}', data: { path: resolved } }`

**5. `executeFilesRename(params)`:**
- Extract: `oldPath` (string, required), `newPath` (string, required)
- Call `safePath(oldPath)` and `safePath(newPath)` for both paths
- Use `fs.rename(resolvedOld, resolvedNew)`
- Return `{ success: true, output: 'Renamed ${resolvedOld} to ${resolvedNew}', data: { oldPath: resolvedOld, newPath: resolvedNew } }`

**Error handling for ALL functions:**
- Wrap each function body in try/catch
- On `ENOENT`: return `{ success: false, output: '', error: 'File not found: ${path}' }`
- On `EACCES`: return `{ success: false, output: '', error: 'Permission denied: ${path}' }`
- On path traversal Error: return `{ success: false, output: '', error: err.message }`
- On any other error: return `{ success: false, output: '', error: err.message }`
- NEVER throw from any exported function

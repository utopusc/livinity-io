Create 3 new hook files following the exact same pattern as use-containers.ts:

**use-images.ts** -- `export function useImages()`:
- `imagesQuery`: `trpcReact.docker.listImages.useQuery(undefined, {retry: false, refetchInterval: 10000})` (10s poll -- images change less frequently than containers)
- `removeMutation`: `trpcReact.docker.removeImage.useMutation({...})` -- onSuccess refetch images + set success actionResult, onError set error actionResult. 3s/5s timeouts to clear.
- `pruneMutation`: `trpcReact.docker.pruneImages.useMutation({...})` -- onSuccess refetch images + set success actionResult with message showing space reclaimed (format bytes: use `(bytes / 1024 / 1024).toFixed(1) + ' MB'` for MB, `(bytes / 1024 / 1024 / 1024).toFixed(2) + ' GB'` for GB, threshold at 1024 MB). onError set error actionResult.
- `actionResult` state: `useState<{type: 'success' | 'error'; message: string} | null>(null)`
- `removeImage(id: string, force?: boolean)`: wrapper calling removeMutation.mutate
- `pruneImages()`: wrapper calling pruneMutation.mutate
- Return: `{images, isLoading, isError, error, isFetching, refetch, removeImage, isRemoving, pruneImages, isPruning, actionResult, totalSize, totalCount}`
- `totalSize`: sum of all image sizes in bytes
- `totalCount`: images.length

**use-volumes.ts** -- `export function useVolumes()`:
- `volumesQuery`: `trpcReact.docker.listVolumes.useQuery(undefined, {retry: false, refetchInterval: 10000})`
- `removeMutation`: `trpcReact.docker.removeVolume.useMutation({...})` -- requires name + confirmName. onSuccess refetch + success message, onError error message.
- `actionResult` state
- `removeVolume(name: string, confirmName: string)`: wrapper calling removeMutation.mutate({name, confirmName})
- Return: `{volumes, isLoading, isError, error, isFetching, refetch, removeVolume, isRemoving, actionResult, totalCount}`

**use-networks.ts** -- `export function useNetworks()`:
- `networksQuery`: `trpcReact.docker.listNetworks.useQuery(undefined, {retry: false, refetchInterval: 15000})` (15s poll -- networks rarely change)
- No mutations (networks are read-only per CONTEXT.md)
- `inspectedNetwork` state: `useState<string | null>(null)` (network ID to inspect)
- `inspectQuery`: `trpcReact.docker.inspectNetwork.useQuery({id: inspectedNetwork!}, {enabled: !!inspectedNetwork, retry: false})`
- `inspectNetwork(id: string)`: sets inspectedNetwork state
- `clearInspect()`: sets inspectedNetwork to null
- Return: `{networks, isLoading, isError, error, isFetching, refetch, inspectNetwork, clearInspect, inspectedNetworkData: inspectQuery.data, isInspecting: inspectQuery.isFetching, totalCount}`

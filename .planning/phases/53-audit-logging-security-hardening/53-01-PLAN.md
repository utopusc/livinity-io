<objective>
Add end-to-end audit logging for all remote device tool executions.

Purpose: Every tool call on a remote device must be logged for accountability and debugging. The audit trail flows: agent local file -> relay pass-through -> LivOS Redis storage -> tRPC query -> UI display.

Output: Agent writes JSON lines to ~/.livinity/audit.log, sends audit events through relay to LivOS, LivOS stores in Redis (capped 1000/device), new tRPC route exposes audit log, UI shows Activity section per device.
</objective>

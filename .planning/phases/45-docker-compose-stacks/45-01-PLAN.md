**stacks.ts** -- Create new file with these domain functions:

1. **`STACKS_DIR`** = `/opt/livos/data/stacks` constant.

2. **`ensureStacksDir(name: string)`** -- Creates `/opt/livos/data/stacks/{name}/` using `import {mkdir, writeFile, readFile, rm} from 'node:fs/promises'` and `import {existsSync} from 'node:fs'`. Use `{recursive: true}`.

3. **`listStacks()`** -- Use dockerode singleton (import from docker.ts: `import Dockerode from 'dockerode'; const docker = new Dockerode({socketPath: '/var/run/docker.sock'})`) to list all containers (`docker.listContainers({all: true})`). Group by `Labels['com.docker.compose.project']` label. For each project, compute status: all containers running = 'running', all stopped = 'stopped', mixed = 'partial'. Return `StackInfo[]` sorted by name. Skip containers with no compose project label.

4. **`deployStack(input: {name: string, composeYaml: string, envVars?: Array<{key: string, value: string}>})`** -- Validate name matches `/^[a-zA-Z0-9][a-zA-Z0-9_.-]*$/`. Call ensureStacksDir(name). Write `docker-compose.yml` to `STACKS_DIR/{name}/docker-compose.yml`. If envVars provided, write `.env` file as `KEY=value` lines to `STACKS_DIR/{name}/.env`. Run `docker compose -p {name} -f {composePath} up -d` via execa `$`. Return `{success: true, message: 'Stack {name} deployed'}`. If compose fails, throw with `[compose-error]` prefix including stderr.

5. **`editStack(input: {name: string, composeYaml: string, envVars?: Array<{key: string, value: string}>})`** -- Same as deploy but check stack dir exists first (throw `[not-found]` if not). Overwrite files, then run `docker compose -p {name} -f {composePath} up -d --remove-orphans`.

6. **`controlStack(name: string, operation: StackControlOperation)`** -- Check stack dir exists. Build compose command: `docker compose -p {name} -f {composePath} {operation}`. For 'up', append '-d'. Run via execa. Return `{success: true, message: 'Stack {name} {operation} completed'}`.

7. **`removeStack(name: string, removeVolumes?: boolean)`** -- Check stack dir exists. Run `docker compose -p {name} -f {composePath} down` (append `--volumes` if removeVolumes is true). After successful down, remove the stack directory (`rm(stackDir, {recursive: true, force: true})`). Return `{success: true, message: 'Stack {name} removed'}`.

8. **`getStackCompose(name: string)`** -- Read and return the compose YAML from `STACKS_DIR/{name}/docker-compose.yml`. Throw `[not-found]` if file doesn't exist.

9. **`getStackEnv(name: string)`** -- Read `.env` from `STACKS_DIR/{name}/.env`, parse into `Array<{key: string, value: string}>`. Return empty array if file doesn't exist.

Use `import {$} from 'execa'` (same pattern as PM2 module). For all execa calls, set `{cwd: stackDir}` to ensure relative paths in compose files resolve correctly.

Error handling pattern: try/catch around execa, throw descriptive errors with `[compose-error]`, `[not-found]`, or `[validation-error]` prefixes.

Export all functions.
